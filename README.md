1- Fontawesome-webfont
<br>
2- AIcons
<br>
3- icons
<br>
4- Anonymous pro

#I3blocks 

1-Install i3blocks with this command:
<pre>yaourt -S i3blocks</pre>

2-Puts i3blocks.conf. in /home/$USER then restart I3wm with $mod+shift+r
